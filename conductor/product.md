# Initial Concept

I want to create an webapp hosted on my personnal server. This webapp will be called "InfoMap" and the objective is to create a worldwide map, with each countries represented on it, with the possibility to click on a country to have a window displaying the main headlines / news. I want to use the pereplexity API to do that.

# Product Definition - InfoMap

## Goals
- Provide real-time news updates for every country.
- Visualize global events on an interactive map.
- Offer a historical archive of news by region.

## Key Features (MVP)
- **Interactive Map:** A 2D or 3D world map where users can click on specific countries.
- **Perplexity API Integration:** Fetching and summarizing the latest headlines using the Perplexity API for the selected country.
- **Information Modal:** A clean interface to display news summaries and headlines when a country is clicked.
- **Secure Access:** Google OAuth2 authentication required to access the application.
- **Usage Control:** Per-user daily quotas for API calls, manageable via an administrator dashboard.

## Target Audience
- Students or educators looking for an interactive world news tool.

## Visual Identity & UX
- **Minimalist and Clean:** A focus on data and high readability.
- **High-tech and Futuristic:** Utilizing dark mode with glowing accents to create a modern feel.

## Supplementary Information
- **Country Context:** Displaying key statistics like population and GDP alongside news.
- **Regional Trends:** Showing social media trends and hashtags to provide a pulse of the region.
