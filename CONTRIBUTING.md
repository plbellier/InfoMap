## How Can I Contribute?

### Reporting Bugs
- **Check existing issues** before opening a new one
- Use a clear and descriptive title
- Include reproduction steps, expected vs. actual behavior
- Add screenshots or GIFs if helpful

### Suggesting Enhancements
- Use a clear and descriptive title
- Explain the motivation and use case
- If applicable, describe the current behavior vs. the desired behavior

### Pull Requests

1. **Fork the repository** and create your branch from `main`
2. **Follow the code style** of the existing codebase
3. **Write clear commit messages**
4. **Test your changes** locally before submitting
5. **Update documentation** if needed

### Development Setup

```bash
# Clone your fork
git clone https://github.com/your-user/InfoMap.git
cd InfoMap

# Set up environment variables
cp .env.example .env
# Edit .env with your API keys

# Run with Docker
docker-compose up --build
```

### Code Style

- **Python**: Follow PEP 8, use type hints where possible
- **TypeScript/React**: Follow existing patterns, use functional components with hooks
- **Commits**: Use clear, descriptive commit messages

### Questions?

Feel free to open an issue with the label `question` if you need help.
